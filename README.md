# PetRescue
PetRescue es una plataforma web diseñada para optimizar la gestión de mascotas, adopciones, refugios, reportes de extravío y participación comunitaria. El sistema integra módulos completos para administración, seguridad, control de procesos y generación de reportes, permitiendo a refugios y usuarios interactuar de manera eficiente y segura.
